11.9 和

ここでは、和型の型の一意性のために型注釈を必要とする構文形式をとっている。

新しい構文形式

\[ \begin{align} \mathrm{t} \ ::= \ & ... \\ & \mathrm{inl \ t \ as \ T} \tag{タグ付け（左）} \\ & \mathrm{inr \ t \ as \ T} \tag{タグ付け（左）} \\ & \mathrm{case \ t \ of \ inl \ x \Rightarrow t \ | \ inr \ x \Rightarrow t} \tag{場合分け} \\ \mathrm{v} \ ::= \ & ... \\ & \mathrm{inl \ v \ as \ T} \tag{タグ付けの値（左）} \\ & \mathrm{inr \ v \ as \ T} \tag{タグ付けの値（右）} \\ \mathrm{T} \ ::= \ & ... \\ & \mathrm{T + T} \tag{和型} \end{align} \]

新しい評価規則

\[ \mathrm{case(inl \ v_0 \ as T_0) \ of \ inl \ x_1 \Rightarrow t_1 \ | \ inr \ x_2 \Rightarrow t_2} \to [\mathrm{x}_1 \mapsto \mathrm{v_0}]\mathrm{t}_1 \tag{E-CaseInl} \] \[ \mathrm{case(inr \ v_0 \ as T_0) \ of \ inl \ x_1 \Rightarrow t_1 \ | \ inr \ x_2 \Rightarrow t_2} \to [\mathrm{x}_2 \mapsto \mathrm{v_0}]\mathrm{t}_2 \tag{E-CaseInR} \] \[ \frac {\mathrm{t}_0 \to \mathrm{t}_0'} { \mathrm{case \ t_0 \ of \ inl \ x_1 \Rightarrow t_1 \ | \ x_2 \Rightarrow t_2} \to \mathrm{case \ t_0' \ of \ inl \ x_1 \Rightarrow t_1 \ | \ x_2 \Rightarrow t_2} } \tag{E-Case} \] \[ \frac {\mathrm{t}_1 \to \mathrm{t}_1'} {\mathrm{inl \ t_1 \ as \ T_2} \to \mathrm{inl \ t_1' \ as \ T_2}} \tag{E-Inl} \] \[ \frac {\mathrm{t}_1 \to \mathrm{t}_1'} {\mathrm{inr \ t_1 \ as \ T_2} \to \mathrm{inr \ t_1' \ as \ T_2}} \tag{E-Inr} \]

新しい型付け規則

\[ \frac {\Gamma \vdash \mathrm{t}_1 : \mathrm{T}_1} {\Gamma \vdash \mathrm{inl \ t_1 \ as \ T_1 + T_2} : \mathrm{T}_1 + \mathrm{T}_2} \tag{T-Inl} \] \[ \frac {\Gamma \vdash \mathrm{t}_1 : \mathrm{T}_2} {\Gamma \vdash \mathrm{inr \ t_1 \ as \ T_1 + T_2} : \mathrm{T}_1 + \mathrm{T}_2} \tag{T-Inr} \] \[ \frac { \Gamma \vdash \mathrm{t}_0 : \mathrm{T_1 + T_2} \qquad \Gamma, \mathrm{x_1:T_1} \vdash \mathrm{t_1:T} \qquad \Gamma, \mathrm{x_2:T_2} \vdash \mathrm{t_2:T} \qquad } {\Gamma \vdash \mathrm{case \ t_0 \ of \ inl \ x_1 \Rightarrow t_1 \ | \ inr \ x_2 \Rightarrow t_2 : T}} \tag{T-Case} \]

演習11.9.1

caseの型付け規則と図8-1のifの型付け規則の類似性に着目せよ。ifは分岐に対して情報を渡さない、caseを縮退した形の一種とみなせる。 この直観を形式化するため、和型とUnitを用いてtrue、false、ifを派生形式として定義せよ。
