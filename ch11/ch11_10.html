11.10 バリアント

一般化したラベル付きの和型。

新しい構文形式

\[ \begin{align} \mathrm{t} \ ::= \ & ... \\ & \mathrm{<l=t> \ as \ T} \tag{タグ付け} \\ & \mathrm{case \ t \ of} \ <\mathrm{l}_i=\mathrm{x}_i> \Rightarrow {\mathrm{t}_i}^{i \in 1..n} \tag{場合分け} \\ \mathrm{v} \ ::= \ & ... \\ & \mathrm{<l=t> \ as \ T} \tag{タグ付けの値} \\ \mathrm{T} \ ::= \ & ... \\ & <{\mathrm{l}_i:\mathrm{T}_i}^{i \in 1..n}> \tag{バリアント型} \end{align} \]

新しい評価規則

\[ \mathrm{case}(<\mathrm{l}_j = \mathrm{v}_j \ \mathrm{as \ T}>) \ \mathrm{of} \ {<\mathrm{l}_i = \mathrm{x}_i> \Rightarrow \mathrm{t}_i}^{i \in 1..n} \to [\mathrm{x}_j \mapsto \mathrm{v}_j]\mathrm{t}_j \tag{E-CaseVariant} \] \[ \frac {\mathrm{t}_0 \to \mathrm{t}_0'} { \mathrm{case \ t_0 \ of} \ {<\mathrm{l}_i = \mathrm{x}_i> \Rightarrow \mathrm{t}_i}^{i \in 1..n} \to \mathrm{case \ t_0' \ of} \ {<\mathrm{l}_i = \mathrm{x}_i> \Rightarrow \mathrm{t}_i}^{i \in 1..n} } \tag{E-Case} \] \[ \frac {\mathrm{t}_i \to \mathrm{t}_i'} { <\mathrm{l}_i = \mathrm{t}_i> \ \mathrm{as \ T} \to <\mathrm{l}_i = \mathrm{t}_i'> \ \mathrm{as \ T} } \tag{E-Variant} \]

新しい型付け規則

\[ \frac {\Gamma \vdash \mathrm{t}_j : \mathrm{T}_j} {\Gamma \vdash <\mathrm{l}_j = \mathrm{t}_j> \mathrm{as} <{\mathrm{l}_i : \mathrm{T}_i}^{i \in 1..n}> : <{\mathrm{l}_i : \mathrm{T}_i}^{i \in 1..n}>} \tag{T-Variant} \] \[ \frac { \Gamma \vdash \mathrm{t}_0 : <{\mathrm{l}_i : \mathrm{T}_i}^{i \in 1..n}> \qquad 各iに対して\Gamma, \mathrm{x}_i : \mathrm{T}_i \vdash \mathrm{t}_i : \mathrm{T} } {\Gamma \vdash \mathrm{case \ t_0 \ of} \ {<\mathrm{l}_i = \mathrm{x}_i>} \Rightarrow \mathrm{t}_i}^{i \in 1..n} : \mathrm{T} \tag{T-Case} \]

オプション

    OptionalNat = <none:Unit, some:Nat>;

列挙型

    Weekday = <monday:Unit, tuesday:Unit, wednesday:Unit, thursday:Unit, friday:Unit>;

単一フィールドのバリアント

    V = <l:T>;
